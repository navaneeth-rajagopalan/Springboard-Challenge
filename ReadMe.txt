Steps to run the Application:
1. Install the latest version of Node.js
2. Unzip the files and save in local disk
3. Open command prompt from the saved location
4. run the command node server.js
5. Note down the port number the server is listening on
6. open the browser and open the url -> localhost:<port# from step 5>

Features:
1. Search by tag name
2. click on course title/ image, the description and sign up link will open up.
